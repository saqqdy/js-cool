# unique

数组去重。

## 用法

```js
import { unique } from 'js-cool'
```

## 签名

```typescript
function unique<T>(arr: T[]): T[]
```

## 参数

| 参数  | 类型  | 描述         |
| ----- | ----- | ------------ |
| `arr` | `T[]` | 要去重的数组 |

## 返回值

`T[]` - 去重后的新数组。

## 示例

```js
unique([1, 2, 2, 3, 3, 4]) // [1, 2, 3, 4]
unique(['a', 'b', 'a', 'c']) // ['a', 'b', 'c']
unique([1, '1', 1]) // [1, '1']
```

## 注意

- 使用严格相等（`===`）进行比较
- 保持首次出现的顺序
- 返回新数组，不修改原数组
