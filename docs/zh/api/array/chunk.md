# chunk

将数组拆分成多个指定大小的块。

## 用法

```js
import { chunk } from 'js-cool'
```

## 签名

```typescript
function chunk<T>(array: T[], size?: number): T[][]
```

## 参数

| 参数    | 类型     | 描述                      |
| ------- | -------- | ------------------------- |
| `array` | `T[]`    | 要处理的数组              |
| `size`  | `number` | 每个块的长度（默认：`1`） |

## 返回值

`T[][]` - 新的分块数组。

## 示例

```js
chunk(['a', 'b', 'c', 'd'], 2)
// [['a', 'b'], ['c', 'd']]

chunk(['a', 'b', 'c', 'd'], 3)
// [['a', 'b', 'c'], ['d']]

chunk(['a', 'b', 'c', 'd'], 1)
// [['a'], ['b'], ['c'], ['d']]
```

## 注意

- 如果输入不是数组或为空，返回空数组
- size 参数会向下取整，最小值为 1
- 最后一个块可能小于指定的大小
